ENG.

Rules:

1. The camera name must follow the following rules:
Projectname_Shotnumber_Startframe_Endframe
Example: 004_0130_101_321 / SC_022a_1001_1077

2. You must specify a storage path.

Example: G:\Praxis_Work_SRO\Publish\Shots\S01\cfx\ep01\SRO_ep01_004_0130\preview\CH_GumiFemale

Features:

The preview scale is generated with the same value as the render settings.
Even if an object is selected, the selection is cleared, and a preview is generated.
It generates a preview following the current project filename.
It overwrites files with the same name.

Planned additions:
Switch from selected camera to persp camera.
